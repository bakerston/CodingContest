## 1047. Remove All Adjacent Duplicates In String
## Stack
## Python
```swift
def removeDuplicates(self, s: str) -> str:
        st = []
        for ch in s:
            if not st or ch != st[-1]:
                st.append(ch)
            else:
                st.pop()
        return "".join(st)
```
