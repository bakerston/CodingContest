## 1470. Shuffle the Array
## Python
```swift
def shuffle(self, nums: List[int], n: int) -> List[int]:
        res=[]
        for x,y in zip(nums[:n],nums[n:]):
            res.append(x)
            res.append(y)
        return res
```

## CPP
```swift
vector<int> shuffle(vector<int>& nums, int n) {
        vector<int> ans(2 * n);
        for (int i = 0; i < n; ++i)
            ans[i * 2] = nums[i];
        for (int i = 0; i < n; ++i)
            ans[i * 2 + 1] = nums[i + n];
        return ans;
    } 
```


## JAVA
```swift
public int[] shuffle(int[] nums, int n) {
        int[] ans = new int[n * 2];
        for (int i = 0; i < n; ++ i)
            ans[i * 2] = nums[i];
        for (int i = 0; i < n; ++ i)
            ans[i * 2 + 1] = nums[i + n];
        return ans;
    }
```
