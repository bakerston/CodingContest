## 1042. Flower Planting With No Adjacent
#### 贪心法，遍历所有点，对于每个当前的点，找出其邻居的染色情况，用未被选用的颜色去染当前点。

```swift
def gardenNoAdj(self, n: int, paths: List[List[int]]) -> List[int]:
      ans, graph, full = [0] * n, collections.defaultdict(set), {1, 2, 3, 4}
      for i, j in paths:
          graph[i - 1].add(j - 1)
          graph[j - 1].add(i - 1)
      for i in range(n):
          curset = set()
          for j in graph[i]:

          res = full - curset
          color[i] = res.pop()
      return color
```
