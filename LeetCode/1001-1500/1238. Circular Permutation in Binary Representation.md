## 1238. Circular Permutation in Binary Representation
## Math
```swift
def circularPermutation(self, n: int, start: int) -> List[int]:
        ans = [0, 1]
        for k in range(1, n):
            new = [x + 2 ** k for x in ans]
            ans = ans + new[::-1]
        idx = ans.index(start)
        return ans[idx:] + ans[:idx
```
