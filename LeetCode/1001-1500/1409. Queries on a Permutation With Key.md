## 1409. Queries on a Permutation With Key
#### 按照顺序，每次找出数组p中，queries[i]的位置，记录这个位置到ans，并且把这个数放到数组的第一位。
```swift
def processQueries(self, queries: List[int], m: int) -> List[int]:
      p, ans = [x for x in range(1, m + 1)], []
      for idx in queries:
          pos = p.index(idx)
          ans.append(pos)
          p = [idx] + p[:pos] + p[pos + 1:]
      return ans
```
