## 1084. Sales Analysis III
~~~SQL
SELECT product_id, product_name
FROM
(
SELECT product_id
FROM Sales
GROUP BY product_id
HAVING SUM(sale_date BETWEEN '2019-01-01' AND '2019-03-31') > 0
AND SUM(sale_date < '2019-01-01' OR sale_date > '2019-03-31') = 0 
) s
LEFT JOIN Product p
USING(product_id)
~~~
