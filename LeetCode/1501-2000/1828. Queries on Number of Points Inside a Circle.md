## 暴力法。

```swift
def countPoints(self, points: List[List[int]], queries: List[List[int]]) -> List[int]:
      m, n, ans = len(points), len(queries), []
      for i in range(n):
          xc, yc, r = queries[i]
          cur = 0
          for x, y in points:
              cur += (x - xc) ** 2 + (y - yc) ** 2 <= r ** 2
          ans.append(cur)
      return ans
```
