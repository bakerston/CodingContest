## 1829. Maximum XOR for Each Query
#### 从右至左，我们能够确定的是每次的结果都应该是```2 ** maxbit - 1```,而一开始参与的是数组中全部的数，每一次计算，都把当前数组中的最后一个数字去掉。
```swift
def getMaximumXor(self, nums: List[int], maximumBit: int) -> List[int]:
      ans, t = [], 2 ** maximumBit - 1
      for num in nums:
          t ^= num
          ans.append(t)
      return ans[::-1]
```
