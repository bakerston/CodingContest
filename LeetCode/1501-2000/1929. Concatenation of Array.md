## 1929. Concatenation of Array
## Python
```swift
def getConcatenation(self, nums: List[int]) -> List[int]:
        return nums * 2
```
## CPP
```swift
public:
    vector<int> getConcatenation(vector<int>& nums) {
        int n = nums.size();
        vector<int> ans(n * 2);
        for (int i = 0; i < n; i++){
            ans[i] = ans[i + n] = nums[i];
        }
        return ans;
    }
```
