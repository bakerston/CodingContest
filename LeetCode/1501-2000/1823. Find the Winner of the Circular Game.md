## 1823. Find the Winner of the Circular Game
#### 用index与当前数组长度的余数来表示候选人。
```swift
def findTheWinner(self, n: int, k: int) -> int:
        f, cnt, index = [i for i in range(1, n + 1)], 0, 0
        while cnt < n - 1:
            index += k - 1
            index %= n - cnt
            f.remove(f[index])
            cnt += 1
            index %= (n - cnt)          
        return f[0]
```
