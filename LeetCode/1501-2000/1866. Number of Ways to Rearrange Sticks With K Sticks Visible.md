## 1866. Number of Ways to Rearrange Sticks With K Sticks Visible
```swift
def rearrangeSticks(self, n: int, k: int) -> int:
      mod = 10 ** 9 + 7

      @cache
      def dfs(n, k):
          if k == n:
              return 1
          if k < 1 or k > n:
              return 0
          return (dfs(n - 1, k - 1) + (n - 1) * dfs(n - 1, k)) % mod

      return dfs(n, k)
```
