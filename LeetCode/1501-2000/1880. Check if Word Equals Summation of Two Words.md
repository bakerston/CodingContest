## 1880. Check if Word Equals Summation of Two Words
```swift
def isSumEqual(self, firstWord: str, secondWord: str, targetWord: str) -> bool:
      def tran(string):
          ans = ""
          for s in string:
              ans += str(ord(s) - 97)
          return int(ans)

      return tran(firstWord) + tran(secondWord) == tran(targetWord)
```
