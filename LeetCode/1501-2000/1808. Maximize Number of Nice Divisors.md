## 1808. Maximize Number of Nice Divisors
## Python
```swift
def maxNiceDivisors(self, n: int) -> int:
        M = 10 ** 9 + 7
        if n <= 3: 
            return n
        if n % 3 == 0: 
            return pow(3, n // 3, M)
        if n % 3 == 1: 
            return (pow(3, (n-4) // 3, M) * 4) % M
        return (2 * pow(3, n // 3, M)) % M
```
