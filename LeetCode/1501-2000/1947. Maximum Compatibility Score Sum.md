## 1947. Maximum Compatibility Score Sum
## Python
```swift
def maxCompatibilitySum(self, S: List[List[int]], M: List[List[int]]) -> int:
        m, n = len(S), len(S[0])
        ans = 0

        for cands in itertools.permutations(list(range(m)), m):
            cur = 0
            for i in range(m):
                cur += sum(x == y for x, y in zip(S[i], M[cands[i]]))
            ans = max(cur, ans)
        
        return ans
```
