## 2130. Maximum Twin Sum of a Linked List

#### Python
```swift
def pairSum(self, head: Optional[ListNode]) -> int:
        ans = []
        while head:
            ans.append(head.val)
            head = head.next
        res = float('-inf')
        for i in range(len(ans)//2):
            res = max(res, ans[i] + ans[-i - 1])
        return res
```
