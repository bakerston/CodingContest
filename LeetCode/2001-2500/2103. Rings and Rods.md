## 2103. Rings and Rods

Brute Force + Hashtable

#### Python
```swift
def countPoints(self, A: str) -> int:
        res = [set() for i in range(10)]
        n = len(A) // 2
        for i in range(n):
            idx = int(A[i * 2 + 1])
            ch = A[i * 2]
            res[idx].add(ch)
        ans = 0
        for i in range(10):
            if len(res[i]) == 3:
                ans += 1
        return ans
```
