## 1. Two Sum
## Python
```swift
 def twoSum(self, A: List[int], t: int) -> List[int]:
        d = collections.defaultdict(list)
        for i, x in enumerate(A):
            d[x].append(i)
        for key in d.keys():
            if t - key in d:
                if t - key != key:
                    return d[key] + d[t - key]
                else:
                    if len(d[key]) == 2:
                        return d[key]
```
