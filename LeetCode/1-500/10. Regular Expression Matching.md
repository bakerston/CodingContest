## 10. Regular Expression Matching
```swift
def isMatch(self, s: str, p: str) -> bool:
        if not p:
            return not s
        fm = bool(s) and p[0] in {".", s[0]}
        if len(p) >= 2 and p[1] == "*":
            return (self.isMatch(s, p[2:]) or fm and self.isMatch(s[1:], p))
        else:
            return fm and self.isMatch(s[1:], p[1:])
```
