## 765. Couples Holding Hands
#### Greedy
```swift
def minSwapsCouples(self, row: List[int]) -> int:
      res = [r//2 for r in row]
      ans = 0
      for i in range(0, len(row), 2):
          j = res.index(res[i], i + 1)
          if j != i + 1:
              res[j] = res[i + 1]
              ans += 1      
      return ans

```
