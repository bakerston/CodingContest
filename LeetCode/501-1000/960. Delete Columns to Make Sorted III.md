## 960. Delete Columns to Make Sorted III
## DP
```swift
def minDeletionSize(self, S: List[str]) -> int:
      m, n = len(S[0]), len(S)
      dp = [1] * m
      for i in range(1, m):
          for j in range(i):
              cur = 0
              for ch in range(n):
                  cur += (S[ch][i] >= S[ch][j])
              if cur == n:
                  dp[i] = max(dp[i], dp[j] + 1)
      return m - max(dp)
```
