## 254. Factor Combinations
## Backtracking
```swift
def getFactors(self, n: int) -> List[List[int]]:
        if n == 1:
            return []
        res = []
        def dfs(path, rest, target):
            if len(path) > 0:
                res.append(path + [target])
            for i in range(rest, int(math.sqrt(target)) + 1):
                if not target % i:
                    dfs(path + [i], i, target // i)
        dfs([], 2, n)
        return res
```
