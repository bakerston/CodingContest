## 732. My Calendar III
## Boundary count, Time complexity O(N ^ 2)
## Python
```swift
def __init__(self):
    self.c = collections.Counter()

def book(self, start: int, end: int) -> int:
    self.c[start] += 1
    self.c[end] -= 1

    cur, ans = 0, 0
    for x in sorted(self.c):
        cur += self.c[x]
        ans = max(ans, cur)

    return ans
```
